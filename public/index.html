Toys api documentation

Routes

All HTTP methods are supported. You can use http or https for your requests.

  • Get 10 toys by page number:/toys?page=number (GET)
  • Search 10 toys by name or info in each page:/toys?page=number&s=nameOrInfo (GET)
  • Get 10 toys by category in each page/toys?s=category (GET)
  • Get toy by id/toys/single/:id (GET)

    • Add toy - (only for register user - must send user's auth-token through headers):/toys (POST)
